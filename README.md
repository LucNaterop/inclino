## Inclino
Inclinometer collision detection script in python

## Usage
Simply run p

```
python inclino.py example.csv
```

where `example.csv` is a csv file containing inclination measurement series as columns, just as `example.csv` looks like:


```
1.163,2.225
1.400,1.675
2.438,0.075
3.688,-1.088
3.575,-1.250
3.400,-1.075
....and so on...
```
